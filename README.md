# Static Website

- A simple static website created for testing and practice purposes.  
- You can use it for deploying static sites on platforms like GitHub Pages, AWS S3, or Nginx.

-How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/patilrahul99/Static-website.git

2. Navigate to the project directory:
   cd Static-website

3. Open index.html in your browser:
   start index.html
